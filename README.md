# Data Cleaning with SQL Server (SSMS)

This project demonstrates how to clean a real-world CSV dataset using SQL Server (SSMS).

## Key Cleaning Steps
- Imported CSV data into SQL Server
- Standardized categorical columns (order_status, product_name)
- Cleaned numeric fields (quantity)
- Applied InitCap logic to customer names
- Removed duplicate records
- Created a final clean dataset using a single atomic query

## Tools Used
- SQL Server Management Studio (SSMS)
- T-SQL

## Output
- Raw table: Orders
- Clean table: Orders_Clean
